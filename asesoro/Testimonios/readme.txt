Un slider de testimonios ligero, con CPT y estilos heredados.

== Description ==

Este plugin permite gestionar y mostrar testimonios de clientes mediante un Custom Post Type.

Incluye un shortcode `[ver_testimonios]` que renderiza un slider moderno y responsivo.

Características:
*   Gestión sencilla desde el admin.
*   Campos para Nombre, Empresa/Cargo y Foto.
*   Slider responsivo sin dependencias pesadas (jQuery opcional).
*   Estilos minimalistas que heredan de tu tema.

== Installation ==

1. Sube la carpeta del plugin al directorio `/wp-content/plugins/`.
2. Activa el plugin desde el menú 'Plugins' de WordPress.
3. Ve a 'Testimonios' y añade algunos testimonios.
4. Usa el shortcode `[ver_testimonios]` donde quieras mostrar el slider.

== Frequently Asked Questions ==

= ¿Puedo cambiar los estilos del slider? =
Sí, el plugin añade un archivo CSS básico, pero puedes sobrescribirlo fácilmente en tu tema hijo o mediante CSS personalizado.
